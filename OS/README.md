## Directory Structure

```
OS/
├── x64/    # Bootable images for x86_64 targets (PC, embedded x86)
└── ARM/    # Bootable images for ARM targets (RPi, CM4, BeagleBone)
```

### `x64/`
Complete bootable OSOlogic Linux images for x86_64 industrial PCs and embedded x86 platforms. Ready-to-flash `.img` files.

### `ARM/`
Complete bootable OSOlogic Linux images for ARM single-board computers (Raspberry Pi 4/5, CM4, BeagleBone). Ready-to-flash `.img` files for SD card or eMMC.
