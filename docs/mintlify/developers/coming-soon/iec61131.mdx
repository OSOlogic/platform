## Overview

OSOlogic is integrating full support for the IEC 61131-3 standard. This will allow automation engineers to program the PLC using familiar, industry-standard languages:
- Structured Text (ST)
- Ladder Diagram (LD)
- Function Block Diagram (FBD)
- Sequential Function Chart (SFC)

This implementation is powered by an integration with the OpenPLC Editor, automatically compiling standard logic into highly optimized C code that executes directly against the Real-Time API.
