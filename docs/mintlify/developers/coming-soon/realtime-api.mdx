## Overview

The Real-Time API is a C/C++ interface that provides direct, lock-free access to the PLC Core's internal I/O buffers. By utilizing POSIX shared memory (`/dev/shm`), it bypasses the MariaDB database completely, enabling sub-millisecond and nanosecond-level latency for critical control loops and external applications.
