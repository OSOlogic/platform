## What it adds

Enterprise-grade security and regulatory readiness on top of the platform's secure-by-design
foundation:

- **Identity** — RBAC with fine-grained permissions, LDAP / Active Directory / SSO (OIDC, SAML).
- **Audit** — tamper-evident audit trails of configuration and control actions.
- **IEC 62443** — compliance packs, hardening baselines, and documentation for certification.
- **Secure boot & signing** — managed device identity, signed images, and key lifecycle.

## How it connects

Security integrates at the platform edges rather than in your control logic: authentication
and authorization wrap the existing OPC-UA, REST, and MQTT interfaces, so Community Edition
applications work unchanged while access becomes centrally governed. The base `secure_state`
safety mechanism remains in the Community Edition; the certified, audited layer is added here.

## Why Enterprise

RBAC/SSO integration, audit infrastructure, and IEC 62443 compliance packs are governed,
supported deliverables provided as an Enterprise add-on.
