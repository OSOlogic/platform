Community contributions, third-party integrations, and experimental extensions to the OSOlogic platform.

## What belongs here

- **Third-party integrations** — drivers, gateways, or SDK examples contributed by the community that have not yet been promoted to a first-class module
- **Hardware BSPs** — board support packages for hardware targets not officially maintained by Roig Borrell S.L. or Ibercomp S.L.
- **Custom function blocks** — reusable IEC 61131-3 function block libraries contributed by the community
- **Tools and utilities** — scripts, diagnostic tools, or helper applications built on top of OSOlogic
