OSOLogic on x86_64 — running.

One docker compose up and you have a live OSOLogic: a MariaDB source-of-truth, the osodb hub, a REST API, an OPC-UA server and every web interface — all wired together. Control anything over the network and watch it move in the UIs below.

The UIs on this page are pre-pointed at this sandbox's REST API (same origin), so they connect out of the box.
