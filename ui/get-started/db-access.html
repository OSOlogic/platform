OSOlogic Tag access · SQL

Read & write PLC tags from any language

OSOLogic is data-centric: the plant is the tags table. With DB credentials, any language that speaks SQL can read a value (SELECT value) and control it with a set-point (UPDATE required_value) — no drivers, no SDKs. The scan applies the set-point and writes the result back.

host localhost (or the PLC IP)  ·  db osodb  ·  port 3306  ·  user osoapp / osoapp
read SELECT value FROM tags WHERE id='hass.sensor.tank_level'  ·  write UPDATE tags SET required_value=1 WHERE id='hass.switch.pump'

Permissions are enforced by the database (MariaDB GRANTs) — read-only users can SELECT but not UPDATE. The same works over the REST API and OPC-UA.
