OSOlogic Home Assistant

Home Assistant gateway

An optional compatibility gateway: through Home Assistant's public API, the devices it already supports (Zigbee, Z-Wave, Matter, MQTT…) appear in osodb as typed tags. Point it at your instance and choose which device domains to import.

Gateway

Reads over HA's public API and honours its licence — it complements Home Assistant, it doesn't replace it. See Gateways and the bridge & mapper.

Import these device domains

Each domain maps via the driver library to typed osodb tags (hass.<domain>.<object>).
