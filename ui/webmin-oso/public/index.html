Webmin-style admin · lightweight flavour

A lightweight admin surface that lives inside an existing Webmin install — for boxes already managed that way. It's an optional integration, separate from the modern Cockpit-style flavour and the standalone OSOlogic modules below.

Optional · on the roadmap

The native-Webmin integration — exposing the OSOlogic modules as panels inside an existing Webmin install — is planned, not shipped yet. You don't need it: the OSOlogic modules below run standalone (no Webmin required), and the Cockpit-style flavour is available now.
