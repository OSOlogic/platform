OSOlogic Structured Text

Structured Text editor

IEC 61131-3 Structured Text — the common intermediate form. Ladder compiles to it, and it feeds the osoST toolchain → p-code → runtime.

Check validates structure (VAR/END_VAR, IF/END_IF, CASE/END_CASE…). Compile to p-code runs on the device via osoST.

Programs are managed in Programs & Scripts; compiled output deploys via PLC Projects.
